# Datacluster Pro (Big Data Analysis Service)

Проєкт складається з двох частин: **Python-бекенду** (для машинного навчання) та **React-фронтенду** (для інтерфейсу користувача). Щоб сервіс працював коректно, обидві частини мають працювати одночасно у двох окремих терміналах.

## Крок 1: Запуск Бекенду (Аналітичне ядро)
Бекенд обробляє файли, тренує алгоритми та здійснює взаємодію з ШІ через Flask.

1. Відкрийте термінал у головній папці проєкту (`d:\UNI\Diploma\practical\big-data-analysis-service`).
2. Активуйте віртуальне середовище:
   - На Windows: `.\venv\Scripts\activate`
3. Запустіть сервер:
   ```bash
   python app.py
   ```
   *Сервер запуститься на порту http://127.0.0.1:5000*

## Крок 2: Запуск Фронтенду (Інтерфейс користувача)
Фронтенд забезпечує графічну панель користувача (Dashboard).

1. Відкрийте **другий** окремий термінал.
2. Перейдіть у папку фронтенду:
   ```bash
   cd frontend
   ```
3. Встановіть залежності (тільки під час першого запуску):
   ```bash
   npm install
   ```
4. Запустіть сервіс розробки:
   ```bash
   npm run dev
   ```
   *UI запуститься та буде доступний за посиланням http://localhost:5173*
   
> **Примітка:** Для демонстрації продукту використовуйте прикріплений файл `sample_dataset.csv`.
